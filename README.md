# welcome to CATPURE THE FLAG AI AGENT {CTF-AGENT}
- This is AI Powered CTF Challenges practice platform.
- you have to solve the challenges based on the SQL injection, prompt injection, XSS, cryptography... etc.
- you have to solve the 15-20 question from the different level easy, medium and hard levels.

## Getting Started

First,
1. clone the repo:

```bash
git clone {repo link}
```

2. put the google api key in <b>.env</b> file.

```bash
GOOGLE_API_KEY = "AI--------------------"
```
3. start the server run this command in a terminal

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
